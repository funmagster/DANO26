# Анализ влияния сервисного сбора на метрики сервиса «Топливо» в Т-Банке

Мы провели анализ данных по сервису «Топливо» — одному из сервисов экосистемы Т-Банка, который позволяет пользователям заправлять автомобиль дистанционно. Основная задача исследования — определить, как размер сервисного сбора влияет на ключевые метрики клиентов и сервиса.

## Описание исследования

В рамках анализа рассматриваются заказы клиентов по неделям, а также дополнительные характеристики пользователей. Особое внимание уделяется влиянию кэшбэку, который пользователь получает за заказ.

**Исследовательский вопрос:**

Зависит ли удержание клиента от кешбэка?

**Гипотеза:**

Эластичность к сервисному сбору максимальна, когда его сумма сопоставима с размером кэшбэк

## Структура репозитория

- **final_analysis.ipynb**  
  Основной ноутбук исследования. Здесь проводится полный цикл анализа:  
  - Предварительный анализ данных  
  - Математическое подтверждение гипотезы о влиянии сервисного сбора  
  - Проверка устойчивости результатов  
  - Визуализация и интерпретация ключевых графиков и выводов

- **basement.ipynb**  
  Дополнительный ноутбук, в котором воспроизводятся вспомогательные графики и зависимости, использованные в приложениях итоговой презентации.
